# Portfolio-Website

A responsive personal portfolio website showcasing my skills, projects, and academic background as a Computer Science student. The website is designed with a modern purple-themed interface, smooth animations, and a fully responsive layout across mobile, tablet, laptop, and desktop devices.

## Features

- Responsive design for all screen sizes
- Interactive navigation with active section highlighting
- Smooth scroll reveal animations
- About, Skills, Projects, and Contact sections
- Functional contact form using FormSubmit
- Resume download functionality
- Social media integration
- Modern UI with custom purple theme (#b711e0)

## Built With

- HTML5
- CSS3
- JavaScript

## Project Structure

```
Portfolio/
│── assets/
│── index.html
│── style.css
│── app.js
└── README.md
```

## Sections

- **About** – Introduction and academic background
- **Skills** – Technical skills and core computer science concepts
- **Projects** – Portfolio projects with GitHub links
- **Contact** – Contact form and social links
